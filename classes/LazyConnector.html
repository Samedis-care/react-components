Class LazyConnector<KeyT, VisibilityT, CustomT>

Forwards all read calls (index, read) to the real connector directly but queues all writing calls (create, update, delete) which can be fired at once

remarks

Does not support relations, enhances read/index calls with locally updated data

  • VisibilityT: PageVisibility
